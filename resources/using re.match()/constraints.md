The 5 Strict Constraints

Extraction: You must extract the <number> and the <word> from every valid line. Ignore all other lines in the file.

Sorting: You must sort the words in descending order based on their numerical ID (highest ID number first).Output 

Formatting: The final output must be a single string where the sorted words are joined by a hyphen (-).

No Third-Party Packages: You cannot use external libraries like pandas, regex (re), or numpy. Use only native Python built-ins.

Robustness: Your code must handle empty lines or spaces gracefully without raising an error.
